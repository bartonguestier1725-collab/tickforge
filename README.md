# TickForge

Open-source tick chart viewer. One HTML file. No install.

Built with [Lightweight Charts](https://github.com/tradingview/lightweight-charts) by TradingView.

## Features

- Real-time candlestick charts from tick data
- 13 timeframes: Tick, 1s, 5s, 15s, 30s, 1m, 5m, 15m, 1h, 4h, 1D, 1W, 1M
- Crypto (BTC/ETH) works instantly — no API key needed
- FX pairs available with API key
- Dark theme, full-screen, auto-reconnect
- Customizable with AI — feed the HTML to Claude Code, Codex, or ChatGPT

## Quick Start

1. Download `index.html`
2. Open in browser
3. Done

Crypto charts start immediately via Binance WebSocket.

## FX Data

FX pairs require an API key.

- International: [Tick Aggregator Demo](https://tick.hugen.tokyo/demo)
- 日本語: [noteメンバーシップ](https://note.com/ishinabegen/membership)
